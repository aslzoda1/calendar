## ✨ Xususiyatlari (Features)

*   🌿 **Yashil Gradient Dizayn:** Ko'zni charchatmaydigan, tabiat va yoshlik energiyasini aks ettiruvchi interfeys.
*   🎭 **Modern UI/UX:** Neon borderlar, 3D shadow effektlar va silliq hover animatsiyalari.
*   🎂 **Avtomatik Tracking:** Tug‘ilgan kunlar va muhim bayramlar yil davomida avtomatik ravishda yangilanib turadi.
*   ⚡ **Tezkor yuklanish:** Maxsus "Loading Animation" orqali foydalanuvchi tajribasi yaxshilangan.
*   📱 **Responsive:** Barcha qurilmalar (Smartfon, Planshet, Desktop) uchun to'liq moslashuvchan.
