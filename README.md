# 📅 Aslzoda’s Calendar — Aesthetic Event Tracker

![Netlify Status](https://img.shields.io/netlify/aslzoda-calendar.netlify.app)
![License](https://img.shields.io/badge/license-MIT-green)
![Tech](https://img.shields.io/badge/tech-HTML5%20%7C%20CSS3%20%7C%20JS-orange)

**Aslzoda’s Calendar** — bu shunchaki taqvim emas, balki foydalanuvchiga yuqori darajadagi vizual tajriba ulashuvchi modern veb-ilova. U bayramlar, maxsus kunlar va tug‘ilgan kunlarni kuzatish uchun maxsus ishlab chiqilgan.

🌐 **Live Demo:** [aslzoda-calendar.netlify.app](https://aslzoda-calendar.netlify.app)

---

## ✨ Xususiyatlari (Features)

*   🌿 **Yashil Gradient Dizayn:** Ko'zni charchatmaydigan, tabiat va yoshlik energiyasini aks ettiruvchi interfeys.
*   🎭 **Modern UI/UX:** Neon borderlar, 3D shadow effektlar va silliq hover animatsiyalari.
*   🎂 **Avtomatik Tracking:** Tug‘ilgan kunlar va muhim bayramlar yil davomida avtomatik ravishda yangilanib turadi.
*   ⚡ **Tezkor yuklanish:** Maxsus "Loading Animation" orqali foydalanuvchi tajribasi yaxshilangan.
*   📱 **Responsive:** Barcha qurilmalar (Smartfon, Planshet, Desktop) uchun to'liq moslashuvchan.

---

## 🛠 Texnologiyalar (Tech Stack)

| Texnalogiya | Vazifasi |
| :--- | :--- |
| **HTML5** | Sahifa strukturasi |
| **CSS3** | Glassmorphism, Neon effektlar va Grid/Flexbox |
| **Evo Calendar** | Taqvim mantiqi va bayramlar kutubxonasi |
| **Google Fonts** | Poppins — zamonaviy va o'qishga qulay shrift |
| **Netlify** | Yuqori tezlikdagi hosting |
